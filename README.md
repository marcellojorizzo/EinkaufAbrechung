Excel Sheet with Macros for Grocery / Shared Household Budget Book
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
///////////////////////////////////////////////////////////////////
How to Use:
Open the Excel File:

Ensure macros are enabled.

Reset and Start:

Click "Abrechnung Starten" to reset all values and start the program.
Steps:

Enter Names:
Input a Name here.
Enter the Amount for the Purchase:
Specify the total amount spent on groceries.

Enter Item Details:
Input the quantity and unit price of each item.

Continue or Stop Adding Items:
The program will ask if you want to add more items. Repeat Step 3 for each new item until no more items need to be added.
Print Option:
The program will ask if you want to print the details. If not, you can continue adding items using the "Artikel einfügen" button.
Adjustments and Manual Entries:

The currency symbols (e.g., 10, 20, 50, 100, 200, or 500 €) are active buttons that set the "Betrag für den Einkauf"
All entries can be manually adjusted as needed.
Printing:

Use the "Formular drucken" button to print the details at any time.
